# Fifteen Hundred Marketing Page

One-page creator partnership landing page for the Fifteen Hundred iOS app.

## Run locally

```bash
npm install
npm run dev
```

## Screenshot placeholders

Replace these files with exported app images when ready:

- `public/screenshots/onboarding-preview.png`
- `public/screenshots/meal-timeline-preview.png`
- `public/screenshots/food-card-preview.png`

The page is static and can be deployed to Vercel or Netlify with `npm run build`.
